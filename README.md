# kubuntu-branding
Brand Guides and Assets

## Overview

This repository contains the current Kubuntu Brand assets, and Guidance for their use.

The portfolio includes :-
 - Style Guides
 - Colour Pallette
 - Font selections
 - Kubuntu Logo (png and svg)
 - Wallpapers
 - Calamares-Installer slides
